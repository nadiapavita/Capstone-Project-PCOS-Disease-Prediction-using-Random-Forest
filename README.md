# Capstone-Project-PCOS-Disease-Prediction-using-Random-Forest


**Live Demo:**  
Try the interactive PCOS prediction system here:  
[https://your-streamlit-link](https://pcos-prediction-kelompok19.streamlit.app/)

This project builds a machine learning model to predict the likelihood of **Polycystic Ovary Syndrome (PCOS)** based on patient health indicators. The model uses the **Random Forest algorithm** and applies the **SMOTEENN resampling technique** to address class imbalance in the dataset.

In addition to the prediction model, the system also includes a **PCOS information chatbot** powered by the Gemini API. The chatbot is restricted to answering questions related to PCOS to ensure relevant and focused responses.

---

## Project Features

- PCOS prediction using **Random Forest**
- Handling imbalanced data using **SMOTEENN**
- Model evaluation using classification metrics
- **PCOS information chatbot** powered by Gemini API
- **Interactive Streamlit web application**
- Dashboard to explore the dataset

---

## Machine Learning Pipeline

Dataset → Data Preprocessing → SMOTEENN → Random Forest Model → Evaluation → Streamlit Application
