## Notebook description

Notebook pada implementasi kali ini menggunakan bahasa Python dimulai dengan memanggil dataset dari kaggle, kemudian dilakukan eksplorasi dan cleaning data, lalu dilakukan SMOTEENN pada dataset berdasarkan hasil dari eksplorasi nya, kemudian diterapkan model Random Forest dan dilakukan evaluasi model, tahap terakhir di lakukan penerapan nya pada Streamlit dan penggunaan Gemini API unuk fitur chatbot.
